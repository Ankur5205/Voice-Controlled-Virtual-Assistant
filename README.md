# Voice-Controlled-Virtual-Assistant
# This project is a Voice Controlled Virtual Assistant developed using Python.
# It uses speech recognition to listen to user commands and text-to-speech to respond verbally.
# The assistant can search Wikipedia, tell the current time, and open websites like Google and YouTube.
# It also supports basic system commands such as opening Notepad and exiting the program.
# This project demonstrates voice interaction, automation, and Python library integration.
